# Phase 236-255 Source-Aware Discovery

Date: 2026-05-19

## What changed

- Shared discovery status mapping is tested for `verified`, `estimated`, `needs_review`, and `unknown`.
- Global search carries source/confidence status where available.
- Module pieces are treated as source-review discovery items instead of silently appearing fully verified.
- Module browse copy now clarifies that compatible cartridge shapes can be shown while some exact pairings remain source pending.
- Guides are labelled as planned and source pending.
- Tier-list public copy now uses "reference" wording and avoids presenting rankings as official claims.

## Policy kept

Unverified values were not marked as verified. No source URLs, expiry dates, rewards, formulas, legal guarantees, or official claims were invented.
